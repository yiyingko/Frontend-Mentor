# Frontend Mentor Challenges

This repository is a collection of my solutions to various **Frontend Mentor** challenges.  
Instead of creating a separate repository for each challenge, I organise them as **individual sub-projects (folders)** inside this main repo for clarity and maintainability.

Each folder represents one challenge and contains its own source code, assets, and (where relevant) a dedicated README with implementation notes.
